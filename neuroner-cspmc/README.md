# neuroner-cspmc

NeuroNER LSTM-CRF engine (Python 3.7 / TensorFlow 1.x).

This directory contains the `neuroner` Python package, its virtualenv, and the **HTTP inference sidecar** in [`sidecar/`](sidecar/) (Dockerfile, `compose.yaml`, `serve.py`). All data, models, and training output live at the project root. See [docs/neuroner-setup.md](../docs/neuroner-setup.md) for setup and usage.
